前端介绍

说好的html呢。。

好吧，其实html只是网页的一部分，而前端才是整个网页。总的来说，前端无外乎3个部分：html,css,javascript。

我们将会分别介绍这三个部分。

html

首先我要说的是，HTML是一种树。这是一个非常典型的树形数据结构在实际中的应用。

文档中栗子不好举啊，我们去菜鸟教程的一个网站 html制作

接下来我们可以尝试进行修改了。只有body里的内容才是我们的编辑内容。

  1. 首先我们介绍元素，元素是《》里的内容，不同的元素里面的内容也有不同的定义。

    div ： 定义代码块，便于管理，主要是便于css管理

    a：定义链接

    p：定义文本

    img：图片

    script：定义script执行脚本

    table：定义表格

    元素就是树中的节点，元素都有开始和结束，在开始和结束之间的其他元素，都是该元素的子孙节点。

  2. 属性

    属性是用来定义节点作用的，是附属于节点的。比如控制节点颜色，字体大小等等。

    id：该属性在整个HTML中只出现一次，用来定义唯一的节点。

    class：该属性出现多次，用来定义属于同一类的节点。

    href：属于a标签，用来表示a标签下文本链接的其他网址。

  3. 轴

    轴指节点之间的关系

    父节点：紧邻的上层节点

    子节点，兄弟节点，孙节点......

    关于HTML ，介绍这么多已经够了。注意理解。

css

css 不多说，是一个界面管理与美化的语言，用于布局。对爬虫没多大用。

javascript

html是不可执行语言，就是一个文本语言，无逻辑。

而javascript 就是用来在网页上执行逻辑处理的一种语言。可执行判断，循环等逻辑操作，让网页更加灵活。

用的比较多的地方有加载数据。填充数据。

HTML有两种形式，静态和动态

静态的HTML是框架和数据都已经写死的一个HTML文件

动态的HTML是只写了框架，没有数据，数据通过javascript添加

如果html是个架子，json是数据，那么javascrapt 可以把数据按照需求填充到HTML适当地方。

浏览器可以执行这个语言
